# OCT-MONITOR-NETWORK-TEMP
Sistema de monitoramento em tempo real (Modo Nativo Win32) para verificação de temperatura de hardware e largura de banda de rede (1Gbps vs 100Mbps) em salas de exames de OCT. Desenvolvido em Python com Tkinter e Win32API para prevenção de danos térmicos e falhas de conectividade.
